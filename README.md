> *Analytics is the bridge between raw data and business success.*

This repository showcases professional-grade, end-to-end machine learning projects focused on business analytics. Each project demonstrates a complete workflow—from data preprocessing and feature engineering to model training and evaluation using `scikit-learn` pipelines.

---

## 📈 Projects Included

### 🩺 1. Healthcare Insurance Cost Prediction
* **Focus:** Linear & Ridge Regression
* **Objective:** Forecast medical insurance costs based on patient demographics.
* **Key Highlights:** `Pipeline` architecture, residual analysis, model performance evaluation ($R^2$, RMSE), and non-linear pattern identification.

### 📊 2. Telco Customer Churn Classification
* **Focus:** Logistic Regression
* **Objective:** Predict customer churn to drive data-informed retention strategies.
* **Key Highlights:** Confusion matrix analysis, probability threshold optimization, and classification metrics.

## 🛠️ Technical Stack
Our systems leverage **`scikit-learn Pipeline`** and **`ColumnTransformer`** architecture to ensure production-ready code structures.

* **Visualization:** `Seaborn`, `Matplotlib`
